# SUPERMERCADOWOOW
Proyecto  final de la clase de componente web
Este sistema incluye backend en Node.js y un fronted moderno en Angular.

# Requisitos
node.js Versión 18 o superior.
base de datos que uilizamos PostgreSQL y DBeaver 25.3.3

# Configuracion
# Backend
1. Entra a la carpeta Backend.
2. Ejecuta npm install para instalar dependencias.
3. Crea un archivo .env siguiendo el ejemplo de .env.example
4. Inicia el servidor con `node src/index.js`.

# Frontend
1. Entra a la carpeta Fronted
2. Ejecuta npm install.
3. Inicia la aplicación con ng serve
4. Abra http://localhost:4200 ahi en la terminal se sale.
